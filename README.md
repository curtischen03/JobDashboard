# Job Dashboard

**Job Dashboard** is a streamlined, full-stack application designed to simplify the job search process for developers. By providing a daily curated list of full-stack positions and robust tracking tools, it helps users stay organized and informed throughout their career journey. Secure authentication via Google ensures a personalized experience for every applicant.

## Key Features

- **Daily Job Updates**: Automatically refreshes with the latest full-stack job openings to ensure you never miss an opportunity.
- **Application Tracking**: Effortlessly manage your pipeline by tracking every applied job.
- **Performance Statistics**: Visual insights and stats on your apply rate and jobs applied.
- **Secure Authentication**: Quick and secure access using Google Auth.
- **Modern UI/UX**: Built with Shadcn/UI for a clean, professional, and accessible interface.

